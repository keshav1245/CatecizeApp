C@TECHIZE - An Overview

This application built for Android Platform is a Robust and Smart Solution for taking Quiz on Electronic device instead of taking on Paper.This app has easy to use UI Design.It features a Login and signup page for authenticated users. The Questions are fetched from Xammp server. Each question has 30 seconds to attempt and answers are shown at that time only.
Results are stored back on Xammp Server which can be downloaded as CSV File in 1 Click !

This Application uses following Technologies :

  1. XAMMP MySQL Server
  2. Firebase Cloud Database
  3. Android Studio
  4. HTML,CSS &PHP
  5. Photoshop
  6. JAVA
  7. XML
